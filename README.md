#OrgoNise---productivity mobile application

OverView-
OrgoNice is a productivity-focused mobile application designed to help users organize daily activities, manage tasks efficiently, and improve time management. The app provides structured planning tools such as to-do lists, goal tracking, and personal productivity features in a simple and user-friendly interface.

Features-
To-do list for managing daily tasks
Study timer to improve focus and productivity
Digital diary for personal notes and tracking
Goal tracking (daily, weekly, monthly)
Mini games for engagement and relaxation
Clean and easy-to-use interface

Tech stack-
HTML
CSS
Java Script
Android studio

Project Purpose-
This project was developed to demonstrate my ability to build a functional mobile application focused on productivity and user experience. It showcases my skills in frontend development, application design, and creating practical solutions for task management and daily organization
